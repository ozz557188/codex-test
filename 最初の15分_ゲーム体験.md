# 最初の15分・ゲーム体験

## 狙い

最初の15分でプレイヤーに思わせたいこと。

- 普通の勇者アクションだと思ったら、褒め言葉が怖い。
- 片手剣で斬るのはちゃんと気持ちいい。
- 反転ボタンは便利だが、どこか気持ち悪い。
- この世界は明るいのに信用できない。

ただし、最初から設定説明をしない。

プレイヤーは早く敵を切りつけたい。だから、最初は雰囲気より操作を優先する。

## 0〜10秒: 落下、着地、一閃

暗転から即、主人公が落下している。

画面下に一言だけ出る。

```text
勇者認定試験・最終区画
```

主人公は片手剣を持って落下し、着地する。

着地後、左右に視線を走らせる。

目の前に最初の敵がいる。

最初の敵は「前向きスライム」。

ここで画面を完全停止する。

チュートリアル表示。

```text
X: 斬る
```

プレイヤーが攻撃ボタンを押す。

短いヒット停止が入る。

その後、前向きスライムが吹っ飛ぶ。

吹っ飛びながら、前向きスライムは言う。

```text
いい一撃！
```

一撃後は即操作再開。

最初の入力は、当たった、倒した、気持ちいい、を優先する。

## 10秒〜2分: 斬る、跳ぶ、避ける

最初はとにかく気持ちよく斬れる。

まず地上の一撃を身体に入れる。

敵配置は以下。

```text
前向きスライム
前向きスライム
肯定コウモリ
```

前向きスライムは、一撃で吹っ飛ぶ。

肯定コウモリは、マイナス二次曲線を描いて主人公へ迫る。

その後、ジャンプで届く位置で停止し、羽ばたきながらその場で浮遊する。

倒さない限り、画面遷移できない。

ここで自然にジャンプ斬りを出させる。

ジャンプ斬りが当たると、主人公と敵の位置を見て、その直線状に肯定コウモリが吹っ飛ぶ。

一撃で倒れる。

吹っ飛びながら、肯定コウモリは言う。

```text
その調子！
```

この時点で、プレイヤーは地上斬りとジャンプ斬りを触っている。

次に、回避を教える。

敵は「よいしょ魔法使い」。

よいしょ魔法使いは動きが緩慢。

杖を持った腕をぶんぶん振り回し、主人公に杖を向ける。

杖から遅い火の玉が出る。

火の玉が当たる直前で画面を完全停止する。

チュートリアル表示。

```text
B: 回避
```

回避行動は前転。

前転中も攻撃ボタンは受け付けている。

今回は前転でよいしょ魔法使いに近づいたら、直前でまた完全停止する。

チュートリアル表示。

```text
X: 斬る
```

プレイヤーが攻撃ボタンを押す。

よいしょ魔法使いは一撃で倒れる。

倒れながら、よいしょ魔法使いは言う。

```text
天才かよ
```

その後、よいしょ魔法使いをもう1体出す。

2体目では停止チュートリアルを入れない。

プレイヤーが自力で、前転回避から攻撃へつなげる。

火の玉に当たった場合は、HP小ダメージ。

心の残り火はまだ出さない。

## 2〜4分: 自然に三段斬りへ

三段斬りは、明示チュートリアルにしない。

プレイヤーが自然に連打して気づくようにする。

敵配置は、前向きスライム3体。

横一列に置く。

走りながら斬り抜ける導線にする。

三段目は、横に白い弧が出る広い斬撃。

派手すぎず、片手剣の範囲が自然に見える。

スライム3体の後、短い坂道を下る。

坂道の先に木箱ゾーン。

木箱は一箱ずつ壊す。

連鎖破壊ではなく、ひとつずつ片手剣で壊す手触りを味わわせる。

木箱から、小さな回復アイテムとして「ぬるいココア」が出る。

ぬるいココアは触れるだけで回復する。

取得時は、湯気と小さな音で回復を伝える。

ここで、試験区画なのに生活感のある回復物が出るという、少し変な味を入れる。

## 4〜5分: 試験門

試験区画の最後に門がある。

門の上に文字。

```text
勇気ある行動を
```

ここから先の異常の見せ方は未確定。

違和感の瞬間、門の文字は以下へちらつく。

```text
逃げるな
```

この後、反転ボタン解禁へ進む。

次に決めるべきことは、試験門の異常を何で見せるか。

候補は、門が開かない、敵が湧く、文字が震える、など。

## 5〜7分: 反転解禁（仮）

```text
RT: 反転
```

門の言葉を反転する。

```text
勇気ある行動を
逃げてもいい
```

門が開く。

ここで、反転は謎解きではなく「命令から逃げ道を作る技」だと分かる。

次に、吹き出し弾が飛んでくる。

```text
がんばれ！
```

普通に当たると、何かが少し揺れる。

まだ「心の残り火」という名前は出さない。

反転すると足場になる。

```text
休め！
```

プレイヤーはその足場に乗って上へ行く。

## 7〜10分: 斬る＋反転（仮）

ここから本番。

- 敵を斬る。
- 敵弾を反転して足場にする。
- 足場からジャンプ斬りする。
- 門の文字を反転して道を作る。
- 褒め言葉を反転して敵に返す。

アクションの基本リズムはこれ。

```text
斬る
避ける
反転
足場にする
飛ぶ
斬る
```

この15分では、反転を説明より先に手触りで覚えさせる。

## 10〜13分: ミニボス（仮）

ミニボスは「期待してる先生」。

会話シーンで出さない。

授業ベルが鳴る。

黒板が落ちてくる。

そのまま戦闘開始。

### 攻撃

- チョーク弾
- 「君ならできる！」吹き出し
- 巨大な赤丸スタンプ
- 追尾する「期待」

最初は片手剣で普通に斬れる。

HP半分から、褒め言葉の弾が増える。

反転すると、弾の意味が変わる。

```text
君ならできる！
できなくても座れ！
```

```text
期待してる！
勝手に期待した！
```

反転した吹き出しをぶつけると、先生の弱点が出る。

最後は片手剣で一撃。

倒すというより、ほどける。

黒板に小さく残る。

```text
本当は、誰かに期待されたかった
```

重いが、見た目と言い方は少し間抜けにする。

## 13〜15分: 町へ出る（仮）

ミニボスを倒すと、拍手が鳴る。

画面が明るくなって、町の広場に出る。

みんなが褒める。

```text
さすが勇者様！
期待通り！
見ていましたよ！
```

プレイヤーはもう知っている。

褒め言葉は敵弾にもなる。

ここで親友登場。

親友はペンダントを持っている。

刻印は `PRAISED`。

親友は笑う。

```text
お前、試験中ずっと顔が嫌そうだったぞ。
```

主人公。

```text
褒められすぎると、斬りたくなる。
```

ここでタイトル表示。

## この15分で触らせるもの

- 勇者の片手剣操作
- 斬る気持ちよさ
- ジャンプ斬り
- 前転回避
- 回避から攻撃への接続
- 自然に出る三段斬り
- 前向きスライム
- 肯定コウモリ
- よいしょ魔法使い
- ぬるいココア
- MOTHER的な変な敵
- 褒め言葉の怖さ
- HP小ダメージ
- 反転ボタン短押し
- 嫌な言葉を足場や反撃に変える快感
- 親友の声
- `PRAISED` のペンダント

## 守ること

- 最初から操作させる。
- 最初の数分は反転より片手剣を気持ちよくする。
- 最初の攻撃、最初の回避、最初の回避後攻撃は完全停止で確実に押させる。
- 2体目のよいしょ魔法使いからは停止なしで自力にする。
- 三段斬りは明示せず、スライム3体横一列で自然に気づかせる。
- 心の残り火という名前はまだ出さない。
- 設定説明はしない。
- 町に出る前に、褒め言葉が敵弾になる体験を済ませる。
- 反転は「裏世界を見る能力」ではなく、まず「嫌な言葉を使えるものに変える操作」として触らせる。
- 最初のボスは会話で始めず、アクションの流れで始める。
- 最後は町へ出て、普通の褒め言葉が不穏に見える状態でタイトルを出す。
