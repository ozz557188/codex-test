# 最初の15分・ゲーム体験

## 狙い

最初の15分でプレイヤーに思わせたいこと。

- 普通の勇者アクションだと思ったら、褒め言葉が怖い。
- 片手剣で斬るのはちゃんと気持ちいい。
- 反転ボタンは便利だが、どこか気持ち悪い。
- この世界は明るいのに信用できない。

ただし、最初から設定説明をしない。

プレイヤーは早く敵を切りつけたい。だから、最初は雰囲気より操作を優先する。

## 0〜10秒: 即操作

暗転から即、主人公が走っている。

画面下に一言だけ出る。

```text
勇者認定試験・最終区画
```

この時点でもうプレイヤーは動ける。

主人公は片手剣を持っている。目の前に敵がいる。

最初の敵は「勝手に拍手する手袋」。

説明なしで斬れる。

斬ると手袋が吹っ飛びながら拍手する。

```text
いい剣筋！
```

ここで、プレイヤーは笑うか、少しだけ嫌な気分になる。

## 10秒〜2分: 斬るだけで気持ちいい一本道

最初はとにかく気持ちよく斬れる。

- 走る
- ジャンプ
- 片手剣三段斬り
- 空中斬り
- 回避
- 敵を倒す
- 木箱を壊す

チュートリアル文字は最低限。

```text
X: 斬る
A: 跳ぶ
B: 回避
```

敵は軽めにする。

- 勝手に拍手する手袋
- やる気のある藁人形
- ほめすぎる看板
- 前向きなスライム

倒すと変な一言を残す。

```text
もっと褒めてから倒して。
君なら壊せると思ってた。
看板にも痛覚はある。
```

MOTHER的な変さはここで出す。

ただしテンポは止めない。変な台詞を読むために足を止めさせすぎない。

## 2〜4分: 片手剣の気持ちよさ

小部屋ごとに片手剣の基礎を触らせる。

- 三段斬りで複数敵を処理する。
- ジャンプ斬りで浮いた敵を落とす。
- 回避で突進を避ける。
- 下り坂で敵を連続で斬る。
- 最後に少し大きい敵を倒す。

ここでは反転をまだ使わせない。

まず「このゲームは普通に斬って気持ちいい」と思わせる。

反転ゲームだからといって、最初から頭を使わせない。プレイヤーはまだ賢くなりたいのではなく、剣を振りたい。

## 4〜5分: 最初の違和感

試験区画の最後に門がある。

門の上に文字。

```text
勇者は前へ進む
```

門の前に敵が湧く。

倒す。

また湧く。

倒す。

また湧く。

プレイヤーは処理できるが、終わらない。

そこで親友の声。

```text
なあ、それ、褒めてるんじゃなくて命令じゃないか？
```

門の文字がちらつく。

```text
勇者は前へ進む
勇者は逃げるな
```

ここで初めて反転ボタンを解禁する。

## 5〜7分: 反転解禁

```text
RT: 反転
```

門の褒め言葉を反転する。

```text
勇者は前へ進む
勇者は戻ってもいい
```

門が開く。

ここで、反転は謎解きではなく「命令から逃げ道を作る技」だと分かる。

次に、吹き出し弾が飛んでくる。

```text
がんばれ！
```

普通に当たると、心の残り火が少し削れる。

反転すると足場になる。

```text
休め！
```

プレイヤーはその足場に乗って上へ行く。

## 7〜10分: 斬る＋反転

ここから本番。

- 敵を斬る。
- 敵弾を反転して足場にする。
- 足場からジャンプ斬りする。
- 門の文字を反転して道を作る。
- 褒め言葉を反転して敵に返す。

アクションの基本リズムはこれ。

```text
斬る
避ける
反転
足場にする
飛ぶ
斬る
```

この15分では、反転を説明より先に手触りで覚えさせる。

## 10〜13分: ミニボス

ミニボスは「期待してる先生」。

会話シーンで出さない。

授業ベルが鳴る。

黒板が落ちてくる。

そのまま戦闘開始。

### 攻撃

- チョーク弾
- 「君ならできる！」吹き出し
- 巨大な赤丸スタンプ
- 追尾する「期待」

最初は片手剣で普通に斬れる。

HP半分から、褒め言葉の弾が増える。

反転すると、弾の意味が変わる。

```text
君ならできる！
できなくても座れ！
```

```text
期待してる！
勝手に期待した！
```

反転した吹き出しをぶつけると、先生の弱点が出る。

最後は片手剣で一撃。

倒すというより、ほどける。

黒板に小さく残る。

```text
本当は、誰かに期待されたかった
```

重いが、見た目と言い方は少し間抜けにする。

## 13〜15分: 町へ出る

ミニボスを倒すと、拍手が鳴る。

画面が明るくなって、町の広場に出る。

みんなが褒める。

```text
さすが勇者様！
期待通り！
見ていましたよ！
```

プレイヤーはもう知っている。

褒め言葉は敵弾にもなる。

ここで親友登場。

親友はペンダントを持っている。

刻印は `PRAISED`。

親友は笑う。

```text
お前、試験中ずっと顔が嫌そうだったぞ。
```

主人公。

```text
褒められすぎると、斬りたくなる。
```

ここでタイトル表示。

## この15分で触らせるもの

- 勇者の片手剣操作
- 斬る気持ちよさ
- 回避とジャンプ
- MOTHER的な変な敵
- 褒め言葉の怖さ
- 心の残り火
- 反転ボタン短押し
- 嫌な言葉を足場や反撃に変える快感
- 親友の声
- `PRAISED` のペンダント

## 守ること

- 最初から操作させる。
- 最初の数分は反転より片手剣を気持ちよくする。
- 設定説明はしない。
- 町に出る前に、褒め言葉が敵弾になる体験を済ませる。
- 反転は「裏世界を見る能力」ではなく、まず「嫌な言葉を使えるものに変える操作」として触らせる。
- 最初のボスは会話で始めず、アクションの流れで始める。
- 最後は町へ出て、普通の褒め言葉が不穏に見える状態でタイトルを出す。
